제작 배경

기업 소개와 제품 특성을 동시에 전달해야 하는 프로젝트로, 자료는 주로 의뢰 내용 + 소개서 형태로 제공되었습니다.

제한된 정보 안에서 “무엇을 강조해야 신뢰가 생기는가”를 기준으로 핵심 메시지를 재정리하고, 촬영/편집까지 단일 기준으로 일관되게 운영하는 것이 목표였습니다.

자료 기반 재구성 기업/제품 동시 전달 톤앤매너 정렬

기획 의도

“회사 소개로 신뢰를 확보하고, 기술을 설명한 뒤 직접 보여주며, 기존 방식 대비 특장점을 명확히 증명한다.”

회사 소개 → 기술 설명 → 실제 시연 → 기존 방식 대비 특장점 제시 순서로 구성해 짧은 러닝타임에서도 전달 흐름이 명확하고 납득이 빠르게 일어나도록 설계했습니다.

메시지 구조화 이해 장벽 완화 적용 맥락 제시

제작 과정

  • 01
    자료 분석 및 리서치 : 전달받은 회사 소개서와 기술 자료를 검토하고 제품 작동 원리와 공정 흐름을 이해하기 위해 관련 리서치 진행
  • 02
    촬영 환경 협의 : 회사와 촬영 가능 범위를 협의하고 공장 촬영 대신 엑스포 전시 부스에서 진행되는 제품 시연 현장 촬영으로 방향 확정
  • 03
    현장 촬영 : 제품 작동 과정을 직접 확인하며 설명을 듣고, 제한된 전시 부스 환경을 고려해 작동 흐름이 잘 보이는 장면 중심으로 촬영 진행
  • 04
    영상 구조 설계 : 기술 이해 흐름이 자연스럽게 이어지도록 회사 소개 → 제품 원리 → 작동 과정 → 특장점 순으로 영상 구조 설계
  • 05
    그래픽 및 편집 : 기존 공정과 제품 적용 후 과정을 As-Is / To-Be 그래픽으로 시각화해 기술 차별점이 직관적으로 전달되도록 편집 진행

연출 포인트

  • 01
    기업 신뢰 기반 도입부 구성 : 기업 로고, 핵심 키워드, 가치 메시지를 먼저 제시해 기업의 기술 정체성과 방향성을 보여준 뒤 솔루션 설명으로 자연스럽게 연결
  • 02
    시연 중심 기술 설명 : 제품 작동 장면을 중심으로 AI 인식, 분석, 분류 과정을 단계적으로 보여주며 기술이 실제 현장에서 어떻게 작동하는지 직관적으로 전달
  • 03
    기술 원리 설명 : 초분광 이미지 기술이 어떻게 물질을 인식하는지 자막과 그래픽을 활용해 설명하며 기술 이해를 보완
  • 04
    공정 통합 구조 시각화 : 기존 선별 공정을 단계별로 제시한 뒤, 여러 선별 과정이 하나로 통합되는 과정을 그래픽 애니메이션으로 표현해 초분광 선별 기술을 통한 공정 압축 구조를 직관적으로 전달
